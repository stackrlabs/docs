# Community Examples [What others are building]

## Official Examples Repository

You can find the official examples repository at [Micro-Rollup examples](https://github.com/stackrlabs/micro-rollup-examples). We also include these as starter templates when initializing a new project using the Stackr CLI.

## Community Examples

### 1. [Ephermal Micro-Rollup](https://github.com/eshaan7/ephemeral-micro-rollup)

an Ephemeral Rollup that can be used as a Scoring System for games like Capture The Flag.

### 2. [Intents Micro-Rollup](https://github.com/Dhruv-2003/IntentsMRU)

Translating intents into on-chain actions enabled by Intents-MRU's decentralized solver network.

### 3. [Repuation System Micro-Rollup](https://github.com/Dhruv-2003/RepuationSystemMRU)

a reputation system developed for the Farcaster platform to address the prevalent issues of fake accounts, spam/bots, and lack of transparency.

### 4. [$AVAIL LST Micro-Rollup](https://github.com/Architsharma7/Avail-LST-MRU)

stAVL - Avail Liquid Staking Tokens

### 5. [Stealth Micro-Rollup](https://github.com/Dhruv-2003/StealthMRU)

A Stealth Address -based privacy enabling micro rollup for all EVM blockchains.

### 6. [UTXO Micro-Rollup](https://github.com/Dhruv-2003/UTXO-rollup)

A UTXO-based transaction model implemented as a micro-rollup.
